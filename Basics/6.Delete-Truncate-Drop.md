# DELETE vs TRUNCATE vs DROP Operation in SQL

## Delete
The **DELETE** statement is used to remove rows from a table based on a specified condition.

```sql
DELETE FROM table_name
WHERE condition;
```

ğŸ”¹ **If you omit the `WHERE` clause, all rows will be deleted!**

---

#### 1. Deleting Specific Rows

```sql
DELETE FROM Employees
WHERE EmployeeID = 101;
```
ğŸ‘‰ This deletes only the row where `EmployeeID` is 101.

---

#### 2. DELETE with JOIN

You can delete records based on a condition from another table using **JOIN**.

```sql
DELETE e
FROM Employees e
JOIN Departments d ON e.DepartmentID = d.DepartmentID
WHERE d.DepartmentName = 'HR';
```
ğŸ‘‰ This removes employees belonging to the **HR department**.

---

#### 3. DELETE All Rows (Without Dropping Table)

```sql
DELETE FROM Employees;
```
ğŸ‘‰ This removes **all records** but keeps the table structure **intact**.
